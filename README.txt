## Compile and Run

### Compiling Main Program
On Windows:
```
compile\main
run\main
```
On Linux:
```
./compile/main
./run/main
```

### Compiling ADT
On Windows:
```
compile\<adt>
run\<adt>
```
example:
```
compile\queue
run\queue
```
On Linux:
```
./compile\/adt>
./run/<adt>
```
example:
```
./compile/queue
./run/queue
```
